# Secure Remote Command Execution System

## Description
A secure client-server system using TCP sockets and SSL/TLS encryption that allows authenticated users to execute remote commands.

## Features
- SSL/TLS secure communication
- Multi-client support using threading
- Authentication using users.txt
- Command execution with restrictions
- Logging system
- GUI client support

## How to Run

### Start Server
python server_ssl.py

### Run CLI Client
python client_ssl.py

### Run GUI Client
python gui_client_ssl.py
